💡 想一想，如果把带有开头零的输入字符串转换成整数，会发生什么？

它们会被删除，因为从数学的角度来看，0005和5是一样的。 所以你只需要找到转换前后的长度差异。

好吧，我应该从什么开始呢？

你应该从将字符串转换为数字开始，并将其分配给一个新的变量。你可以这样做，只是在字符串前加上+ ！ 

let num = +text;

它可以把所有的零从头开始去掉。

接下来该怎么做？

现在你可以立即检查，num 是否等于0 。如果是这样，意味着整个初始字符串由零组成，你可以用string.length 属性返回其长度。 

if (num == 0) // or (!num)
    return text.length;

如何找到新数字的长度？

你需要将数字转换为字符串，并找到其长度属性。用+"" 操作进行转换! 

(num+"").length

那么，我的最终解决方案应该是什么样子的呢？

你只需要将你的代码包装成函数表达式，并返回两个长度之间的差异。

下面是整个解决方案：

function beginningZeros(text: string): number {

    let num = +text;
    if (!num)
        return text.length;
    
    return text.length - (num+"").length;
}

当然，抓住它!

使用正则表达式在字符串中找到特定的模式（从头开始的零或更多的零），并返回符合该模式的子字符串的长度。 

function beginningZeros(t: string): number {
    
    return t.match(/^0*/)[0].length
}

在循环的定义中，声明一个变量并增加它，直到这个索引处的字符串的字符等于 "0"。 

function beginningZeros(text: string): number {
    
    for (var i = 0; text[i] === '0'; i++);

    return i;
}

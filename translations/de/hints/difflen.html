Die letzte Frage. Wie wird meine endgültige Lösung aussehen?

Hier ist der gesamte Code. 

def beginning_zeros(a: str) -> int:

    a_num = int(a)
    if not a_num: # case a as all zeros

        return len(a)

    return len(a) - len(str(a_num))

In dieser Lösung entfernen Sie die Nullen am Anfang der Zeichenkette auf eine andere Weise. 

def beginning_zeros(number: str) -> int:

    return len(number) - len(number.lstrip('0'))

Hier iterieren Sie über die Zeichenkette, vergleichen jedes Zeichen mit "0" und erhöhen im Falle der Gleichheit den zuvor deklarierten Zähler. 

def beginning_zeros(number: str) -> int:

    result = 0
    for n in number:
        if n!= "0":
            break
        result += 1

    return result

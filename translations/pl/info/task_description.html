Dany jest string składający się wyłącznie z cyfr. Sprawdź, ile zer("0") znajduje się na jego początku.

Wejście: String skłądający się z cyfr.

Wyjście: Int.

Przykład:

{% if interpreter.slug == "js-node" %} 
beginningZeros('100') == 0
beginningZeros('001') == 2
beginningZeros('100100') == 0
beginningZeros('001001') == 2
beginningZeros('012345679') == 1
beginningZeros('0000') == 4
{% else %} 
beginning_zeros('100') == 0
beginning_zeros('001') == 2
beginning_zeros('100100') == 0
beginning_zeros('001001') == 2
beginning_zeros('012345679') == 1
beginning_zeros('0000') == 4
{% endif %}

Warunek wstępny: 0-9

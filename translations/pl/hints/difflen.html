proszę pomyśleć, co się stanie, jeśli przekonwertujemy ciąg wejściowy z początkowymi zerami na liczbę całkowitą?

Zostaną one usunięte, ponieważ z matematycznego punktu widzenia 0005 to to samo co 5. Trzeba więc znaleźć różnicę w długości przed i po konwersji.

Jakiej funkcji szukam?

Do rozwiązania tego zadania będzie Pan potrzebował takich funkcji jak str.len(), str(), int().

Proszę przekonwertować wejściowy ciąg znaków na liczbę całkowitą (proszę użyć funkcji int()), przekonwertować z powrotem na ciąg znaków (str()), aby móc porównać długości. Proszę znaleźć (str.len()) i porównać długość ciągu wejściowego i ciągu otrzymanego po konwersji.

Ok. Jak mogę tego użyć?

input_string = '0005'

input_string_len = len(input_string)
# you will get 4

input_string_to_int = int(input_string)
# you will get 5

input_string_to_int_to_str = str(input_string_to_int)
# you will get '5' without zeros

input_string_to_int_to_str_len = len(input_string_to_int_to_str)
# you will get 1

input_string_len - input_string_to_int_to_str_len
# find and return difference

Proszę nie zapomnieć o przypadku specjalnym jako ciąg wszystkich zer.

Ostatnie pytanie. Jak będzie wyglądało moje ostateczne rozwiązanie?

Oto cały kod. 

def beginning_zeros(a: str) -> int:

    a_num = int(a)
    if not a_num: # case a as all zeros

        return len(a)

    return len(a) - len(str(a_num))

def beginning_zeros(number: str) -> int:

    return len(number) - len(number.lstrip('0'))

Tutaj proszę iterować po łańcuchu, porównując każdy znak z "0" i zwiększając wcześniej zadeklarowany licznik w przypadku równości. 

def beginning_zeros(number: str) -> int:

    result = 0
    for n in number:
        if n!= "0":
            break
        result += 1

    return result

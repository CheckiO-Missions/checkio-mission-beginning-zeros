Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

proszę pomyśleć, co się stanie, jeśli przekonwertujemy ciąg wejściowy z początkowymi zerami na liczbę całkowitą?

Zostaną one usunięte, ponieważ z matematycznego punktu widzenia 0005 to to samo co 5. Proszę więc znaleźć różnicę w długości przed i po konwersji.

Ok, od czego powinienem zacząć?

Powinien Pan zacząć od przekonwertowania ciągu na liczbę i przypisania go do nowej zmiennej. Może Pan to zrobić, aby dodać + przed ciągiem! 

let num = +text;

Usunie to wszystkie zera z początku.

Co robić dalej?

Teraz można od razu sprawdzić, czy num jest równe 0. Jeśli tak, oznacza to, że cały początkowy ciąg składa się z zer i można zwrócić jego długość za pomocą właściwości string.length. 

if (num == 0) // or (!num)
    return text.length;

Jak znaleźć długość nowej liczby?

Proszę przekonwertować liczbę z powrotem na ciąg znaków i znaleźć jego długość. Proszę wykonać konwersję za pomocą operacji +""! 

(num+"").length

Jak powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć kod w wyrażenie funkcyjne i zwrócić różnicę między dwiema długościami.

Oto całe rozwiązanie:

function beginningZeros(text: string): number {

    let num = +text;
    if (!num)
        return text.length;
    
    return text.length - (num+"").length;
}

Jasne, proszę pobrać!

Proszę użyć wyrażenia regularnego, aby znaleźć określony wzorzec w ciągu (zero lub więcej zer od początku) i zwrócić długość podciągu, który pasuje do wzorca. 

function beginningZeros(t: string): number {
    
    return t.match(/^0*/)[0].length
}

Wewnątrz definicji pętli proszę zadeklarować zmienną i zwiększać ją do momentu, aż znak ciągu przy tym indeksie będzie równy "0". 

function beginningZeros(text: string): number {
    
    for (var i = 0; text[i] === '0'; i++);

    return i;
}

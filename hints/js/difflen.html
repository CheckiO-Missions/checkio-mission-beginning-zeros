They will be deleted, because from the point of math 0005 is the same is 5. So you just have to find difference in length before and after conversion.

Ok, what should I start from?

You should start from converting the string into number and assign it to a new variable. You may do it, just to add + before the string! 

let num = +text;

It removes all zeros from the beginning.

What to do next?

Now you may immediately check, if the num is equal 0. If it is so, it means the whole initial string consists of zeros and you may return its length with string.length property. 

if (num == 0) // or (!num)
    return text.length;

How to find the length of the new number?

You need to convert the number back to the string and find its length property. Do the conversion with +"" operation! 

(num+"").length

So, how my final solution should look like?

You just need to wrap your code into function expression and return the difference between two lengths.

Here is the whole solution:

function beginningZeros(text: string): number {

    let num = +text;
    if (!num)
        return text.length;
    
    return text.length - (num+"").length;
}

Use regular expression to find specific pattern in the string (zero or more zeros from the beginning) and return the length of substring that fits the pattern. 

function beginningZeros(t: string): number {
    
    return t.match(/^0*/)[0].length
}

Inside the definition of loop declare a variable and increase it until the character of string at this index equal "0". 

function beginningZeros(text: string): number {
    
    for (var i = 0; text[i] === '0'; i++);

    return i;
}
